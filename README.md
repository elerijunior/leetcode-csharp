LeetCode C#

Repositório com soluções de problemas do LeetCode desenvolvidas em C#, com foco na prática de algoritmos, estruturas de dados e resolução de problemas.

Objetivo

Utilizar os desafios do LeetCode como ferramenta de estudo para:

Praticar conceitos fundamentais de algoritmos e estruturas de dados;
Desenvolver raciocínio lógico e capacidade de resolução de problemas;
Consolidar conhecimentos em C# e .NET;
Registrar a evolução técnica ao longo do tempo.
Estrutura
LeetCode/
├── Easy/
├── Medium/
└── Hard/

Cada problema possui sua própria pasta contendo:

Implementação da solução em C#;
Anotações sobre a abordagem utilizada;
Análise de complexidade, quando aplicável.
